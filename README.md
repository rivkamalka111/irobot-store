# 🛒 iRobot Store

אתר חנות למכירת מוצרי iRobot, כולל עמודי מוצרים, עגלת קניות ומערכת הזמנות.

## 🚀 תכונות עיקריות
- הצגת מוצרים עם תמונות ומחירים
- הוספת מוצרים לעגלת הקניות
- חישוב מחיר כולל
- שמירת פריטים בעגלה גם לאחר רענון הדף

## 🛠️ טכנולוגיות
- HTML
- CSS
- JavaScript

## 📌 איך להפעיל?
1. הורידו את הקבצים.
2. פתחו את `index.html` בדפדפן.
